Poisson Image Blending program

This program is a project for Pr. Adrian Munteanu's Image Processing course in Master 1 Bruface (Brussels Faculty of Engineering).

It implements an image blending algorithm based on the solving of Poisson equations. The algorithm is based on the paper "Poisson Image Editing" from P. Prez, M. Gangnet, and A. Blake.
